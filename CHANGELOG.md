## [1.1.0] - 2024-12-24

### 🎉 项目重命名和多语言支持

#### ✨ 重大变更
- ✅ **项目重命名**：从 "Cursor Memory Manager" 更名为 **"CodeMind"**
- ✅ **多语言支持**：新增 6 种语言界面支持
  - 🇨🇳 **简体中文** (默认)
  - 🇺🇸 English
  - 🇹🇼 繁體中文
  - 🇯🇵 日本語
  - 🇰🇷 한국어
  - 🇷🇺 **Русский** (俄文，新增)

#### 🔧 技术改进
- ✅ 使用 VS Code 国际化 API (package.nls.json)
- ✅ 所有命令和配置项支持多语言
- ✅ 自动根据 IDE 语言设置切换界面

## [1.1.0] - 2024-12-24

### 🎉 新增热门语言和框架支持

#### ✨ 新增语言支持（3种）
- ✅ **Kotlin** - Android 开发主流语言，支持 build.gradle.kts 检测
- ✅ **Swift** - iOS/macOS 开发语言，支持 Package.swift 检测
- ✅ **Dart** - Flutter 开发语言，支持 pubspec.yaml 检测

#### ✨ 新增框架支持（16种）
- ✅ **Flutter** - 跨平台移动/桌面应用框架
- ✅ **React Native** - 跨平台移动应用框架
- ✅ **NestJS** - Node.js 企业级后端框架
- ✅ **SvelteKit** - 现代前端框架
- ✅ **SolidJS** - 高性能前端框架
- ✅ **Remix** - 全栈 Web 框架
- ✅ **Nuxt.js** - Vue 全栈框架
- ✅ **Koa** - Node.js 轻量级 Web 框架
- ✅ **Gin** - Go Web 框架
- ✅ **Fiber** - Go Express 风格框架
- ✅ **Echo** - Go 高性能 Web 框架
- ✅ **Rocket** - Rust Web 框架
- ✅ **Actix-web** - Rust 高性能 Web 框架
- ✅ **Laravel** - PHP 现代 Web 框架
- ✅ **Rails** - Ruby Web 框架
- ✅ **ASP.NET Core** - C# Web 框架

#### 🔧 增强功能
- ✅ 增强 Go 项目扫描，自动检测 Gin、Fiber、Echo、Gorilla Mux 框架
- ✅ 增强 Python 项目扫描，支持 Tornado、aiohttp 检测
- ✅ 增强 Rust 项目扫描，支持 Rocket、Actix-web、Tokio 检测
- ✅ 增强 C# 项目扫描，支持 ASP.NET Core 检测
- ✅ 增强 PHP 项目扫描，支持 Laravel、Symfony、CodeIgniter 检测
- ✅ 增强 Ruby 项目扫描，支持 Rails、Sinatra 检测

#### 📊 统计
- **编程语言**：从 11 种增加到 **14 种**
- **框架支持**：从 11 种增加到 **20+ 种**
